Friends of Our Project

We are proud to be part of a vibrant open source community. Here are some amazing open source projects we admire and collaborate with:

Core Open Source Projects

  • SymbiFlow
    An open source FPGA toolchain enabling next-generation development workflows.
  • YosysHQ
    A collection of tools and libraries for open synthesis and digital design.
  • OpenROAD Project
    Automated digital layout tools for VLSI design using open source methods.

Other Cool Projects

  • LiteX
    A Python-based SoC builder for FPGAs, supporting many CPUs and peripherals.
  • TimVideos / HDMI2USB
    Open source HDMI capture solutions for conference recording and streaming.

If you have suggestions for more open source friends to add here, feel free to open an issue!
